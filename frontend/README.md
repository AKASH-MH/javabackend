# Job Portal Frontend

This is the React-based frontend for the Job Portal API.

## Features
- Modern, Dark Mode UI
- Real-time Candidate Registration
- One-click Job Applications
- Live Feedback/Toasts

## Prerequisites
- Node.js (v18+)
- Backend running on `http://localhost:8081`

## Setup & Run

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173`.

## UI Overview
- **Header**: Branding and Candidate Status.
- **Job Grid**: Interactive cards for all open positions.
- **Registration**: Form for new candidates to sign up and start applying.
